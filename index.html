Las Torres de la Madrugada

Experimento audiovisual de música electroacústica generativa y redes estelares paramétricas. Nueve torres de telecomunicaciones y nueve nodos celestiales forman una red de energía sonora.

  • Torres / Estrellas — haz clic para activarlas y conectar la red
  • Láseres — la densidad de conexiones determina el nivel de sobrecarga y muta los timbres
  • Balizas — el pulso rítmico de las luces no activas forma el tejido rítmico base
  • Deriva — velocidad de traslación del firmamento; también modula el decay del reverb
  • CLR — desactiva todos los nodos simultáneamente
1–5 nodos: tensión baja · láseres rojos 6–11: energía media · láseres cyan · FM distortion 12–18: sobrecarga crítica · láseres violeta · arpegio acelerado
Afinación: C minor 11  ·  Tone.js + Vite github ↗
